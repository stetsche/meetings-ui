# meetings-ui

## Project setup
```
npm install
```
## Run storybook
```
npm run storybook
```
## Run dev server
```
npm run serve
```
## Run json server
```
npm run json-server -w data/db.json
```
### Add vue components at
```
src/compnents
```
### To display components in storybook, add stories at  
```
stories
```
### Lints and fixes files:
```
npm run lint
```
